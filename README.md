# Lore – Redes de Proteção
## _Landing Page institucional moderna, responsiva e otimizada para conversão._

### Sobre o Projeto

Este repositório contém o código-fonte da landing page da Lore – Redes de Proteção, incluindo HTML, CSS e JavaScript otimizados para performance, boas práticas e responsividade.

O objetivo é apresentar serviços, benefícios, projetos realizados e canais de contato, garantindo uma experiência clara e direta para o usuário.

### Tecnologias Utilizadas

- HTML5 – Estrutura semântica otimizada
- CSS3 – Layout responsivo com Grid, Flexbox e media queries
- JavaScript Vanilla – Carrossel responsivo, eventos e ajustes dinâmicos
- Imagens otimizadas – Foco em performance


### Funcionalidades

- Header fixo com navegação suave
- Hero responsivo com CTA principal
- Grade de benefícios e seções otimizadas
- Carrossel responsivo com transição fluida
- Ajustes automáticos de layout para tablets e mobile
- Rodapé com informações de contato e CTA final
